# GardenApp 🌱

Мобильное приложение для мониторинга здоровья растений с использованием датчиков Bluetooth Low Energy (BLE). Оно отслеживает экологические показатели, такие как увлажнение, освещенность, температура, влажность и уровень заряда батареи, чтобы поддерживать растения в оптимальном состоянии.

## 🚀 Как это работает

GardenApp подключает беспроводные датчики, размещенные на ваших растениях, для сбора данных в реальном времени. Приложение отображает панель управления с текущими показателями, историческими графиками роста и системой сканирования Bluetooth для настройки новых устройств.

### Основные функции

- **Панель управления в реальном времени**: Визуализация текущих показателей растений
- **Исторические графики**: Отслеживание роста и тенденций со временем
- **Сканирование Bluetooth**: Обнаружение и подключение ближайших датчиков BLE
- **Локальная база данных**: Хранение данных с использованием SQLite для автономной работы
- **Интуитивный интерфейс**: Современный дизайн с навигацией по вкладкам

## 📦 Установка

### Предварительные требования

- Node.js (версия 18 или выше)
- npm или yarn
- Expo CLI
- Мобильное устройство с Android/iOS или эмулятор

### Шаги установки

1. **Клонировать репозиторий**
   ```bash
   git clone <url-репозитория>
   cd GardenApp
   ```

2. **Установить зависимости**
   ```bash
   npm install
   ```

3. **Запустить приложение**
   ```bash
   npx expo start
   ```

4. **Запустить на устройстве**
   - Отсканировать QR-код с помощью приложения Expo Go
   - Или использовать эмулятор/симулятор
   - Для нативной разработки: `npm run android` или `npm run ios`

## 🏗️ Архитектура

### Структура проекта

```
GardenApp/
├── app/                    # Страницы (маршрутизация на основе файлов с Expo Router)
│   ├── (tabs)/            # Навигация по вкладкам
│   │   ├── index.tsx      # Главная панель управления
│   │   ├── explore.tsx    # История и графики
│   │   └── settings.tsx   # Настройки и сканирование BLE
│   ├── bluetooth-onboarding.tsx  # Руководство по подключению
│   └── _layout.tsx        # Корневой макет
├── components/            # Переиспользуемые компоненты
│   └── PlantHealthCard.tsx # Карточка показателей
├── constants/             # Конфигурации и темы
├── db/                    # База данных SQLite
│   └── index.ts          # Конфигурация и заполнение
└── assets/               # Изображения и ресурсы
```

### Используемые технологии

- **Фреймворк**: React Native с Expo SDK 54
- **Навигация**: Expo Router (маршрутизация на основе файлов)
- **База данных**: SQLite с expo-sqlite
- **Bluetooth**: react-native-ble-plx для связи BLE
- **UI/UX**: React Native Reanimated для анимаций
- **Стилизация**: StyleSheet с LinearGradient и Material Icons
- **Типизация**: TypeScript
- **Шрифты**: Google Fonts (Inter, Manrope)

### База данных

Приложение использует SQLite для локального хранения со следующими таблицами:

- **sensors**: Информация о подключенных устройствах
- **metrics**: Экологические показатели (увлажнение, температура и т.д.)
- **history**: Запись событий и действий

При первом использовании данные инициализируются образцами значений.

## 🔧 Разработка

### Доступные команды

```bash
npm start          # Запустить сервер разработки
npm run android    # Запустить на Android
npm run ios        # Запустить на iOS
npm run web        # Запустить в браузере
npm run lint       # Запустить ESLint
```

### Конфигурация Bluetooth

Приложение требует разрешений Bluetooth для работы. На Android разрешения запрашиваются автоматически. Для iOS убедитесь, что разрешения настроены в `app.json`.

### Темы и цвета

Цвета определены в `constants/theme.ts` в соответствии с рекомендациями Material Design 3.
